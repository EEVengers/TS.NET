# Setups

## Standalone

Standalone setup, up to 4 channels.

![Standalone](images/standalone-setup.png)

## Reference clock input

10 MHz reference clock (3.3V, square, 50Ω at source) from external device or lab distribution amplifier.

![Lab reference clock](images/reference-clock-input-setup.png)

## Reference clock output

10 MHz reference clock (3.3V, square, 50Ω at source) output.

![Lab reference clock](images/reference-clock-output-setup.png)

## External trigger source

External device generating trigger pulses (3.3V, square, 50Ω at source). [not yet supported]

![External trigger source](images/external-trigger-source-setup.png)

## External trigger source and reference clock input

External device generating trigger pulses (3.3V, square, 50Ω at source) & 10 MHz reference clock (3.3V, square, 50Ω at source). [not yet supported]

![External trigger source](images/external-trigger-source-and-reference-clock-setup.png)

## Two unit sync

Two units on the same PC, up to 8 channels. [not yet supported]

![2 unit sync](images/2-unit-sync-setup.png)
